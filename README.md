# Analisi di Mercato Orologi di Lusso e Prezzi Usati Italia (IT-01)

Questo repository funge da modello frontend specializzato per una piattaforma di analisi di mercato focalizzata sul settore degli orologi di lusso in Italia (Rolex, Omega, Cartier).

## Funzioni Principali
* **Monitoraggio Prezzi in Tempo Reale**: Analisi dell'andamento dei prezzi per modelli iconici come Submariner, Daytona e Speedmaster sul mercato italiano.
* **Guida al Mercato dell'Usato**: Checklist professionali per l'acquisto e la vendita sicura di orologi certificati in Europa.
* **Rapporti di Investimento**: Dati dettagliati sulla conservazione del valore e previsioni di mercato per i principali marchi svizzeri.
